# Using MyBatis with Spring Boot

### History of commands

```bash
# Install SDKMAN
curl -s "https://get.sdkman.io" | bash

# Install JDK
sdk install java

# Install maven
sdk install maven

# Start containter with MySQL server on 3306
docker compose up -d

# Create mybatis_spring database accessing:
# http://localhost:8081 with credentials root:password
```
